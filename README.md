# Lets Go - quick start template

## Используемые технологии:
* Gulp 4.0
* Pug(Jade)
* Sass(Scss)
* JS+jQuery

## Для старта:
1. install node.js
2. clone this repo
3. cd path/to/
4. npm i gulpjs/gulp-cli -g  // Install the latest Gulp CLI tools globally
5. npm i
6. run "gulp" command to start
